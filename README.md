
# Grid AI
Grid AI is an open-source tool designed to bridge local document data with the power of the Gemini API. The system indexes local PDF files and uses generative AI to provide precise answers and summaries based on the provided content.

Core Stack
Backend: Python with Flask.

AI Integration: Gemini API (Google Generative AI).

Frontend: HTML5 and Vanilla JavaScript.

Data Handling: Local PDF processing and segment indexing.

Features
Gemini Integration: Leverages the Gemini API for natural language understanding and response generation.

Local PDF Indexing: Processes and segments local documents to provide context for the AI.

Hybrid Workflow: Combines the privacy of local data indexing with the advanced reasoning capabilities of cloud-based LLMs.

Clean UI: A minimalist, high-contrast interface built with HTML and JS for a professional user experience.


Required Python libraries: flask, google-generativeai, and PDF processing modules.
